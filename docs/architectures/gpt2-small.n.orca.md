# architecture Gpt2Small

> Decoder-only transformer with learned token + positional embeddings, 12 pre-norm blocks, and a tied language-modeling head. Generated by `n-orca hf convert` from a Hugging Face config.

## hyperparameters

| Name       | Type  | Default |
|------------|-------|---------|
| vocab_size | int   | 50257   |
| d_model    | int   | 768     |
| n_layer    | int   | 12      |
| n_heads    | int   | 12      |
| d_ff       | int   | 3072    |
| max_pos    | int   | 1024    |
| dropout    | float | 0.1     |

## tensors

| Name      | Shape              | Dtype   |
|-----------|--------------------|---------|
| input_ids | (B, S)             | int64   |
| logits    | (B, S, vocab_size) | float32 |
